Atualmente está vendo a seção de administração de respostas do PHPSURVEYOR.

Daqui é possível ver e modificar a ordem das respostas para determinada pergunta em particular.

Botões:

Mover a resposta para cima: Clique aqui para mudar a posição da resposta para cima.

Mover a resposta para baixo: Clique aqui para mudar a posição da resposta para baixo.

Guardar as mudanças: Clique aqui para guardar as mudanças realizadas na resposta.

Excluir resposta: Clique aqui para excluir esta resposta.

Adicionar resposta: Clique aqui para adicionar uma nova resposta.

Corrigir Ordem: Clique aqui para corrigir qualquer problema que puderas ter com a ordem das respostas.
