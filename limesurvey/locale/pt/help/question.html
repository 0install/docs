Atualmente está vendo a seção de administração de perguntas do PHPSURVEYOR.

A partir daqui poderá ver e modificar esta pergunta em particular. Se à sua pergunta podem-se adicionar perguntas individuais, também se pode conseguir isso a partir desta tela. Também podem-se estabelecer condições de como se deve mostrar a pergunta ou não ao clicar no ícone condições.

Botões:

Modificar Pergunta: Clique neste ícone para modificar os detalhes da pergunta atual.

Excluir Pergunta: Clique neste ícone para apagar a pergunta atual. Este botão aparecerá somente se a pergunta não tem respostas.

Exportar Pergunta: Clique neste ícone para exportar a pergunta, incluindo todas as respostas, para um arquivo.

Copiar Pergunta: Clique neste ícone para fazer uma cópia da pergunta atual, incluindo todas as respostas, dentro do mesmo questionário.

Aplicar Condições: Clique neste ícone para usar o formador de condições para estabelecer condições a esta pergunta.

Respostas: Clique neste ícone para adicionar/modificar e apagar a ordem das respostas da pergunta.

Resposta

Responder
Clique neste ícone para adicionar, editar, mover ou apagar respostas da pergunta.

Este ícone só estará disponível quando a pergunta seja do tipo 'Lista', 'Múltiplas Opções’ ou 'Arranjo'. Este tipo de pergunta requer que uma série de respostas sejam introduzidas.

Condições
Clique no ícone para estabelecer, modificar ou apagar condições que possam ser aplicadas à pergunta.

Este ícone mostrará a janela 'Criador de Condições', que permite estabelecer se a pergunta deve ser exibida ou não. Ao aplicar uma condição a uma pergunta, significa que a pergunta NÃO SERÁ MOSTRADA AO MENOS QUE ESSA CONDIÇÃO SE CUMPRA.
