Package zeroinstall

source code

The Python implementation of the Zero Install injector is divided into five sub-packages:

  • zeroinstall.cmd contains one module for each 0install sub-command (providing the shell-scripting interface)
  • zeroinstall.injector contains most of the interesting stuff for managing feeds, keys and downloads and for selecting versions
  • zeroinstall.zerostore contains low-level code for handling the implementation cache (where unpacked packages are stored)
  • zeroinstall.gtkui contains code for making GTK user-interfaces
  • zeroinstall.support contains helper code (not really specific to Zero Install)
