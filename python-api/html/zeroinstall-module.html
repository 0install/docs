Package zeroinstall

source code

The Python implementation of the Zero Install injector is divided into three packages:
  • zeroinstall (this package) just defines a couple of exceptions and the version number.
  • zeroinstall.injector contains most of the interesting stuff.
  • zeroinstall.zerostore contains low-level code for handling the implementation cache.
